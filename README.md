Emma is an open-source gesture control system built with Python, OpenCV and MediaPipe.

The project allows users to control their computer using hand gestures through a webcam: moving the cursor, clicking, scrolling, dragging windows, switching apps and launching custom software.

Emma was created as an experimental interface inspired by futuristic human-computer interaction systems, with the goal of making computer control more natural, accessible and extensible.


Emma es un sistema open source de control por gestos desarrollado con Python, OpenCV y MediaPipe.

Permite controlar la computadora usando la cámara y movimientos de la mano: mover el cursor, hacer click, scrollear, arrastrar ventanas, cambiar de aplicación y ejecutar programas personalizados.

El proyecto nace como una interfaz experimental inspirada en sistemas futuristas de interacción humano-computadora.
